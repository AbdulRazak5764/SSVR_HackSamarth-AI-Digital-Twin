# 🧠 AI Digital Twin for Human Risk Prediction

## 🚀 Overview

AI Digital Twin is a preventive AI system that creates a virtual representation of an individual to predict future risks across multiple domains:

- ❤️ Health Risk
- 💰 Financial Risk
- ⚠ Scam Vulnerability

Unlike traditional systems that react after problems occur, this platform provides early warnings using Machine Learning and Explainable AI.

---

## 🎯 Problem Statement

Individuals cannot effectively assess their future health, financial, and scam-related risks.  
Most current systems are reactive — they respond only after damage has already occurred.

There is no unified, personalized AI platform that predicts multi-domain risks proactively.

---

## 💡 Proposed Solution

We developed an AI-powered Human Digital Twin that:

- Collects user lifestyle and behavioral inputs
- Performs intelligent data analysis
- Predicts risk scores using ML models
- Provides explainable insights
- Displays results in a real-time dashboard

---

## ⚙ Installation Guide

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/ai-digital-twin.git
cd ai-digital-twin
````

### 2️⃣ Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## 🌍 Deployment

### Frontend (Vercel)

1. Push project to GitHub
